## 项目背景

学习并实践 Claude Code——Anthropic 的 AI 编程助手。通过在真实项目中反复使用，探索 AI Agent 在数据分析、代码开发和项目管理中的最佳实践。

目前正在做的事：
- 使用 Claude Code 完成 Amazon 图书数据可视化分析
- 搭建 AI 协作开发标准流程（分析→编码→审查→沉淀）
- 用 Git 管理所有代码和文档，保持可追溯

## 已形成的资产

- **CLAUDE.md** — 项目级 AI 行为约束与工作流定义
- **Memory 体系** — 项目经验、用户背景、反馈偏好的结构化沉淀
- **Skills/Agents 注册表** — 可复用的 AI 能力单元
- **个人网站** — 用 Claude Code 辅助 Next.js 开发

## 学习方向

- 如何写出清晰、可被 AI 正确执行的指令
- 如何在 AI 辅助下保持代码质量和项目控制力
- 如何将 AI 工作流标准化为可复制的流程
