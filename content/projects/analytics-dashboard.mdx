## 个人数据分析 Dashboard

用 Next.js 搭建的个人数据分析看板，整合多个项目的数据指标，实现一站式监控和分析。

### Dashboard 模块

- **项目概览看板** — 各项目关键指标一目了然
- **技能成长追踪** — 按时间维度追踪技能覆盖度变化
- **学习投入分析** — 按领域/工具拆解学习时间分布
- **产出效率看板** — 追踪文章发布、代码提交、项目交付节奏

### 技术亮点

- 全栈 Next.js 实现，前端 Chart.js 可视化
- 响应式设计，手机端友好
- 数据层预留 API 接口，支持扩展
