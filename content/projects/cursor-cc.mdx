## AI 开发工作区

基于 Claude Code + Cursor 搭建的个人 AI 开发工作区，支撑多个项目的日常开发。

### 工作区能力

- Claude Code Agent 工作流 — 自动化重复任务、代码审查
- 统一的项目模板和配置管理 — 新项目启动时间从 2h 降至 10min
- 跨项目代码复用和知识沉淀

### 已支撑项目

- 个人主页网站（Next.js + Tailwind + shadcn/ui）
- 数据分析 Dashboard
- 多个 Python 脚本和自动化工具
