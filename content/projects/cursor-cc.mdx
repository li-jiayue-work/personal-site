## ai-workspace（干活的）

9 目录 monorepo 结构的工作区，`active/` 放活跃项目、`archive/` 存已完成项目、`agents/` 管 Agent 和 Workflow——每个目录按职责划分。根目录 `CLAUDE.md` 定义全局规范，在任何子目录下执行 Claude Code 时自动加载上下文。

**产出**：个人网站 · 两个 Amazon 数据分析项目 · 30+ HTML 模板库

## ai-workflow-notes（记录的）

沉淀 AI 辅助开发的协作方法论，把每次成功的实践固化为可复用的标准。

**包含**：CLAUDE.md 编写规范 · Workflow 流程定义 · 六层知识蒸馏体系 · 架构决策记录（ADR）
