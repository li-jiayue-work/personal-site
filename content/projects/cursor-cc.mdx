## 工作区架构

这是一个结构化的多项目工作区，按生命周期和用途将代码分为 9 个目录：

```
cursor-cc/
├── active/          ← 活跃开发中的项目
├── archive/         ← 已完成或暂停的项目
├── experiments/     ← 实验和原型代码
├── templates/       ← 项目模板和配置模板
├── agents/          ← 自定义 Agent、MCP 配置、Workflow 定义
├── snippets/        ← 跨项目可复用的代码片段
├── learning/        ← 学习笔记
├── assets/          ← 图片、素材等静态资源
└── memory/          ← 跨会话持久记忆
```

## 活跃项目

### beautiful-html-templates

精美的 HTML 模板库，含完整的模板文件、截图预览、运行时脚本和索引系统。

### impeccable

Claude Code Skill 开发框架，包含 Skill 定义（SKILL.md）、自定义 Agent、参考文档和执行脚本，用于构建可复用的 AI 能力单元。

## Agent & Workflow 体系

- **agents/mcp/** — MCP 服务配置，连接外部工具和数据源
- **agents/workflows/** — 自定义工作流定义，标准化任务执行流程

## 开发原则（来自 CLAUDE.md）

1. **先思考，再动手** — 明确假设，展示权衡，不悄悄选择
2. **简洁第一** — 用最少的代码解决问题，不做揣测性开发
3. **外科手术式的修改** — 只碰必须改的，只清理自己造成的烂摊子
4. **目标驱动的执行** — 定义可验证的成功标准，循环执行直到通过
