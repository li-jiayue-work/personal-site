## 电商用户行为分析系统

基于模拟电商数据集构建的完整用户行为分析系统，覆盖从数据清洗到商业洞察的全流程。

### 分析模块

- **GMV 趋势分析** — 按日/周/月维度拆解 GMV 变化趋势
- **用户留存分析** — 计算次日/7 日/30 日留存率，定位流失节点
- **用户分群（RFM）** — 基于消费行为将用户分为高价值/潜力/流失三类
- **转化漏斗分析** — 从浏览 → 加购 → 下单 → 支付的全链路漏斗
- **热销品类排名** — 按品类/时间段交叉分析

### 技术实现

- SQL 完成核心数据查询和聚合（12 条分析查询）
- Python 进行数据清洗和统计分析
- Excel/Tableau 构建可视化 Dashboard（4 个看板）
- AI 辅助生成商业洞察报告

### 核心成果

- 定位出高价值用户占比 18%，贡献 62% GMV
- 发现次日留存率低于行业基准 5pct，提出 3 条改善建议
- 输出一份完整的增长分析报告
